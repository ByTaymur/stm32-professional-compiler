# Changelog

All notable changes to STM32 Professional Compiler will be documented in this file.

## [1.0.0] - 2024-11-18

### Added
- Initial release
- 6 optimization profiles (O0, O1, O2, O3, Os, Og)
- Real-time variable monitor with live editing
- CubeMX project integration
- OpenOCD debugging support
- ST-Link, J-Link, CMSIS-DAP programmer support
- Memory usage visualization
- One-click build and flash
- Sidebar with quick actions
- Project setup wizard
- Configuration options for GCC and OpenOCD paths

### Supported
- All STM32 families (F0-F7, G0-G4, H7, L0-L5, U5, WB, WL)
- VS Code 1.80.0 and later
- VSCodium compatible
