# STM32 Professional Compiler - Packages

This directory contains compiled extension packages.

## Installation

Install the latest version:

```bash
code --install-extension packages/stm32-professional-compiler-1.0.0.vsix
```

Or in VS Code:
1. Open Extensions panel (Ctrl+Shift+X)
